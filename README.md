# 🕵️ Git Profile Detective

A modern and responsive GitHub profile analyzer that helps users explore GitHub profiles with detailed statistics, repositories, followers, following, and account information using the GitHub REST API.

## 🚀 Features

- 🔍 Search any GitHub username
- 👤 View complete user profile information
- 📊 Display public repositories, followers, and following
- 📅 View account creation date
- 📍 Display bio, location, company, and website
- ⭐ Show top repositories
- 📱 Fully responsive UI
- ⚡ Fast performance with Vite
- 🎨 Modern interface built with Tailwind CSS
- ❌ Handles invalid usernames gracefully

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ankit20092005/github-profile-detective.git
```

Navigate to the project

```bash
cd git-profile-detective
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:5173
```
